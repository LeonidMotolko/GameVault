# Game Vault

Android Kotlin final project.

The application keeps a small personal collection of games.
It supports two fixed object types: Video Game and Board Game.

Features:
- Android Studio project
- Kotlin source code
- multiple screens with toolbars
- list screen
- detail screen
- add and edit screen
- delete action
- persistent storage through SharedPreferences
- portrait and landscape layouts
- edge-to-edge enabled on the main screen
